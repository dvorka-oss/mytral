How to obtain your Strava API credentials:

  1. Log in to strava.com.
  2. Go to Settings / My API Application.
  3. Create an API application if you have not done so yet. Set the Authorization Callback Domain to your hostname e.g. 127.0.0.1 for local use.
  4. Copy the Client ID (a short number) and generate / copy the Client Secret (a long hex string).
  5. Paste both values into the form below and click Save Secrets.

Your secrets are stored encrypted on disk and are never displayed again after saving.
